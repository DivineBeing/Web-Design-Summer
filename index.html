Web Development Journey

In this class, I am learning various web development skills, including how to write well-formed HTML5 code. This involves using semantic tags like header, main, and footer to structure the content properly.

Some of the techniques we are focusing on include ensuring our code is standards-compliant and validating it using the W3C Validator. This helps in creating websites that are accessible and functional across different browsers and devices.

Right now, I feel most of this is hard but by the end of this course, I aim to build a fully functional website that showcases my skills and the knowledge I have gained throughout the course.
